Create a Repo

Create a repo in Github, Pagoda, or Springloops. Navigate to your sites folder then: 

$ git clone git@github.com:Abban/git-cheat.git

To create a custom folder name: 

$ mkdir 'foldername'
$ git clone git@github.com:Abban/git-cheat.git foldername

To clone into the current folder you use: 

$ git clone git@github.com:Abban/git-cheat.git .
